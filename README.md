# Edulink-Central

Edulink-Central is an education-focused platform designed to centralize learning content, academic tools, and user interaction in one system. The project aims to make educational resources easier to access and manage through a modern web application.

## Features
- User-friendly interface
- Centralized academic or learning content
- Structured navigation for educational resources
- Scalable full-stack project structure
- Database-backed functionality using Supabase

## Tech Stack
- TypeScript
- React
- Supabase
- PostgreSQL
- HTML
- CSS

## Purpose
This project was developed as part of my practical software development work to strengthen my skills in frontend development, backend integration, and database-driven application design.
